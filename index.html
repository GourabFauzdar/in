    Projects

    Optimal Locally Repairable Codes over small fields for Distributed Storage Systems

    The demand for large-scale data storage is increasing tremendously. As the storage nodes are individually unreliable, redundancy has to be introduced in the system in order to improve the reliability against node failures. Aim to design optimal locally repairable codes over small fields for repairing multiple failed nodes by accessing only local nodes (nearby nodes) in a large-scale distributed storage systems and thereby allowing high data availability.

    Railway Track Security System

    With the development of the high-speed railways, speed and capability of the trains constantly improved, and the traffic density gets more and more serious. The derailments of trains are mainly due to cracks in the railway tracks. Therefore, there is an immense need for crack detection and security system. Aim to design a system to detect a crack in the railway track and alert the nearby railway station via SMS using GSM communication protocols.
